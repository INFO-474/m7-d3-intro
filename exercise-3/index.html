Welcome to Module 7 Exercise 3

To complete this exercise, edit your main.js file. You'll demonstrate a strong familiarity with D3 data binding. Rather than provide you with a set of step-by-step instructions, below you'll find a list of requirements that describe the behavior of the applicaiton you're building.

  • Write a reusable draw function that takes in a dataset as a parameter and properly manages the data-join process between the dataset and the DOM, including enter, update, and exit. You'll have to perform a data-join for both rect and text elements. The width of your rect elements should simply correspond to the score on each exam (we'll go beyond this in later modules).
  • In your HTML or JavaScirpt code, make a button group (buttons should correspond to data for exam1, exam2, and exam3), and assign click events to each button to properly display the corresponding data. Think about efficient ways to assign these events (hint: you only need to do one selection...).
